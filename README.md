# Yuki's Tech Note

Personal tech blog by Yuki Sato.

**Homepage:** `https://yukisato.dev`

## What’s in this repo

- Posts live in `_posts/` (published) and `_drafts/` (drafts)
- Images live in `assets/img/posts/`
- Theme: `jekyll-theme-chirpy` (Jekyll 4.x)
- Deployment: GitHub Pages

## Run locally

```bash
./tools/run.sh
```

Production mode:

```bash
./tools/run.sh -p
```
